This add-on adds a dialog to enter hyperlinks (see screenshot below) and some context menu entries for handling hyperlinks. The shortcuts to insert a hyperlink (default Ctrl+Shift+H) and unlink (Ctrl+Shift+Alt+H) can be configured using the configuration window of 2.1: Tools>Add-ons, select the add-on and click on "Config". Use this at your own risk.
